## Installation instructions

### [docker command](/wiki/%E9%83%A8%E5%B1%AC%E6%AD%A5%E9%A9%9F)

```
$ bundle install
$ yarn install
$ rails db:migrate
$ ./bin/dev
```

- Ruby version

```
ruby '3.1.2'
gem 'rails', '~> 6.1.7'
```

- System dependencies

`tailwind css v2.2.17 for compatibility with rails 6`
